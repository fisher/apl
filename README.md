# The keyboard

## arythmetics

* C-]   ← assign
* '+ -'  add, substract
* C-=   × multiplication
* C-[   ÷  1/x / division
* '*'    exponential / power
* C-S-8 ⍟ natural / specific base logarythm
* C-S-[ ⌹ matrix inverse / division
* C-o   ○ pi / sin / cos / tan

## unsorted yet

* | abs / residue (division remainder)
* C-s ⌈ ceiling / maximum
* C-d ⌊ floor / minimum
* C-b ⊥ decode
* C-n ⊤ encode
* C-S-/ ⊣ left
* C-/ ⊢ right
