# The keyboard

## arythmetics

| Key   | Sym | Name/Function                     |
|:------|:---:|:----------------------------------|
| C-]   |  ←  | assign                            |
|       | + - |  add, substract                   |
| C-=   |  ×  | multiplication                    |
| C-[   |  ÷  | 1/x / division                    |
|       |  *  |   exponential / power             |
| C-S-8 | ⍟  | natural / specific base logarythm |
| C-S-[ | ⌹  | matrix inverse / division         |
| C-o   |  ○  | pi / sin / cos / tan              |
